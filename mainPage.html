Next Generation Terminal Lug for Aircraft Power Systems

Project By: Brandon Williamson, Robert Thompson, Colin McBride, Landen Caldwell

Introduction

The design of the next generation terminal lug for aircraft power systems is a multidisciplinary, partnership project with Gulfstream that will span multiple semesters and across multiple teams. This project aims to redesign how power systems are grounded and connections are made in a way that is ethical and economical.
